## Key bindings

| Key | Command | Description |
| --- | ------- | ----------- |
| b | move | move south west |
| j | move | move south |
| n | move | move south east |
| h | move | move west |
| l | move | move east |
| y | move | move north west |
| k | move | move north |
| u | move | move north east |
| > | move | move down |
| < | move | move up |
| f | fire | fire a wielded item |
| t | throw | throw an item |
| g | get | get an item |
| d | drop | drop an item |
| q	| consume | quaff an item |
| e | consume | eat an item |
| w | wield | wear an item |

## Key bindings

| Key | Command | Description |
| --- | ------- | ----------- |
| b | move | Move south west. The actor will try to move in the south west direction |
| j | move | Move south. The actor will try to move in the south direction |
| n | move | Move south east. The actor will try to move in the south east direction |
| h | move | Move west. The actor will try to move in the west direction |
| l | move | Move east. The actor will try to move in the east direction |
| y | move | Move north west. The actor will try to move in the north west direction |
| k | move | Move north. The actor will try to move in the north direction |
| u | move | Move north east. The actor will try to move in the north east direction |
| > | move | Move down. The actor will try to move down some stairs |
| < | move | Move up. The actor will try to move up some stairs |
| f | fire | Fire a wielded item. The actor will fire a wielded item in a direction, at another actor, at an item, or at a position |
| t | throw | Throw an item. The actor will throw an item in a direction, at another actor, at an item, or at a position |
| g | get | Get/pickup an item. The actor will pickup an item at the position where the actor is positioned |
| d | drop | Drop an item. The actor will drop an item from the inventory at the position where the actor is positioned |
| q	| consume | Quaff/drink an item. The actor will consume a drinkable item from the inventory or from the position where the actor is positioned |
| e | consume | Eat an item. The actor will consume an eatable item from the inventory or from the position where the actor is positioned |
| w | wield | Wear/wield/equip an item. The actor will wear/wield/equip an wearable item from the inventory or from the position where the actor is positioned |
